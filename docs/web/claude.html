Getting Started

# Install VoiceMode
curl -LsSf https://astral.sh/uv/install.sh | sh
uvx voice-mode-install

# Add to Claude
claude mcp add --scope user voicemode -- uvx --refresh voice-mode

For platform-specific setup, see the Getting Started Guide.
