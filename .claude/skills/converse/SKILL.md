# /voicemode:converse

Start a voice conversation using the `mcp__plugin_voicemode_voicemode__converse` MCP tool.

Pass the user's message. All parameters have sensible defaults.

If the MCP tool is unavailable, run `/voicemode:install` to set up VoiceMode.

For full documentation, load the `voicemode` skill.
